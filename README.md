# DSA Journey 🚀

This repository is my personal Data Structures and Algorithms (DSA) learning journal. The goal of this repo is not just to solve problems, but to document the thought process behind each solution and track my progress over time.

Every problem in this repository is accompanied by explanations, approaches, and observations that helped me understand the underlying concepts. By maintaining this record, I aim to build strong problem-solving skills and create a reference that I can revisit whenever needed.

## What You'll Find Here

* 📚 Solutions to DSA problems from various topics.
* 📝 Explanations of the approach and logic used.
* 🔍 Step-by-step understanding of problems whenever applicable.
* 🔄 Multiple implementations of the same problem to compare language-specific approaches. ( Python and CPP )
* 📈 A consistent record of my DSA practice and learning journey.

## Languages Used

I will be solving problems in the following languages:

* **C++** – for understanding low-level concepts, performance, and competitive programming practices.
* **Python** – for concise implementations and improving problem-solving fluency.

## Getting Started

If you'd like to explore or run the solutions locally, clone the repository:

```bash
git clone <repository-url>
cd <repository-name>
```

### Running C++ Programs

Make sure you have a C++ compiler installed (such as `g++`).

```bash
g++ filename.cpp -o output
./output
```

On Windows:

```bash
g++ filename.cpp -o output
output.exe
```

### Running Python Programs

Ensure Python is installed on your system.

```bash
python filename.py
```

or

```bash
python3 filename.py
```
